Yoyaku Manager is for managing events and bookings.

== Description ==
Yoyaku Manager is a WordPress plugin that allows you to manage events and event reservations. It can be used in industries such as lectures, consulting, seminars, schools, networking events, online conferences, and counseling.

It's simple to use, just create an event from the admin screen and then add a shortcode to the article where you want to embed the form. When you publish an article, a reservation form will be rendered depending on the event content. There is also a shortcode to create a form for customers to cancel reservations.

There is no limit to the number of events you can create or the number of emails you can send. It can be used free of charge regardless of the size of your business.

This plugin use Wordpress REST API.

= Yoyaku Manager Pro =

There is also Yoyaku Manager Pro, which has more features than Yoyaku Manager. The Pro version provides an online payment function using Stripe, Google Calendar integration, Zoom meeting integration, adding/deleting notifications, etc.

= Main Features =
* Create/edit/delete events.
* Allow you to create tickets for adults, children, general participation, VIP participation, etc. according to each event.
* Allow you to set an upper limit on the number of tickets (number of participants) for an event.
* Allow you to set the maximum number of tickets purchased per person.
* Customize the forms that are displayed to your customers. (Telephone number, name ruby, gender, postal code, address, date of birth hidden, required or optional)
* Customer management features.
* Reservation management function.
* Shortcode to generate event booking form.
* Shortcode to generate event cancellation form.
* Log function for sent emails.
* Ability to manage reservation approval, hold, cancellation, non-approval emails and periodic notification emails.
* Price display settings according to each country's currency.
* SMTP mail.

= Integrations =
* reCAPTCHA
* Stripe
* Google Calendar & Google Meet
* Zoom

== Installation ==
1. Go to Plugins > Add New within your dashboard.
2. Search for “Yoyaku Manager”.
3. Click install and activate Yoyaku Manager.
